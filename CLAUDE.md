# easy-reviews

## Project Overview
A tool to make code reviews easier and more efficient.

## Tech Stack
- Runtime: Bun
- Language: TypeScript

## Development

### Setup
```bash
bun install
```

### Run
```bash
bun dev
```

### Test
```bash
bun test
```

## Conventions
- Use Bun for all JS/TS tooling
- Follow the ohong Engineering Philosophy (see global CLAUDE.md)
- Keep modules small and focused with clear interfaces
